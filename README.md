# Network-Security-With-ML
Machine learning system which will be able to predict whether a website is phishing or not.

## Data Source
The dataset for this project can be obtained from the UCI Machine Learning Repository:
- **Dataset Name**: Phishing Websites Dataset
- **Link**: [UCI ML Repository - Phishing Websites Dataset](https://archive.ics.uci.edu/ml/datasets/phishing+websites)
- **Alternative Link**: [Kaggle - Phishing Website Dataset](https://www.kaggle.com/datasets/eswarchandt/phishing-website-detector)

### Dataset Description
The dataset contains features extracted from phishing and legitimate websites, including:
- URL-based features
- Domain-based features
- HTML and JavaScript-based features
- External services features

The target variable indicates whether a website is phishing (-1), suspicious (0), or legitimate (1).
